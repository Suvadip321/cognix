# Cognix

An agentic AI research assistant that turns a topic into a full research report.

You enter a topic. Cognix plans research questions, searches the web, evaluates and summarizes sources, and generates a structured markdown report — all in real time. After the report is ready, you can ask follow-up questions answered using the researched material.

---

## Status

In active development

---

## Tech Stack

**Backend** — FastAPI, PostgreSQL, pgvector, LangGraph, LangSmith

**Frontend** — Next.js, TypeScript, Tailwind CSS, shadcn/ui

**AI** — Groq (LLM), OpenAI Embeddings, Tavily (web search)

---

## Features

- Agentic research workflow with planning, search, evaluation, and report generation
- Real-time progress streaming via SSE
- Session-based RAG chat on researched material
- Resume previous research sessions and continue chatting
